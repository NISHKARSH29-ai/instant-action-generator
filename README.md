# Instant Action Generator

## What this is
This is a simple AI tool that gives you ONE clear task to do next
when you feel stuck, overwhelmed, or undecided.

No planning. No motivation. Just action.

---

## How to use
1. Open ChatGPT
2. Copy the prompt below
3. Answer the 3 questions
4. Do the task immediately

---
